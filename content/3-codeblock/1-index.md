## Подсветка удалённого/вставленного кода

Ниже приведен кодовый блок с diff. Строки с + выделены зеленым цветом, указывая на добавление. Строки с - выделены красным оттенком, указывающим на удаление.

```javascript
- const data = ['1','2'];
+ const data = [1,2];
```
